# pyinspect-cli

quick tool to show practical code in terminal

---

## install

```bash
pip install pyinspect-cli
```

---

## use

list all:

```bash
pyinspect-cli list
```

get any practical:

```bash
pyinspect-cli 1
pyinspect-cli 2
pyinspect-cli 3
```

it will just give the access of exact code
